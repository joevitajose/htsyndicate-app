# HTSyndicate Dashboard

## 👋 If a friend asked you to set this up

Read **[FOR-YOUR-FRIEND.md](./FOR-YOUR-FRIEND.md)** — it's a complete brief written for you. ~1 hour of work to get them live.

## 👋 If you're a non-technical user trying to do it yourself

Read **[PLAYBOOK.md](./PLAYBOOK.md)** — long step-by-step guide.

## What this is

A complete React dashboard for an Indian sales/operations company — pipelines, finance, attendance, tasks, automations. Built with Vite + React, no UI library. ~2,000 lines in `src/App.jsx`.

## Files

- `FOR-YOUR-FRIEND.md` — handoff brief for a technical friend (start here if you're helping someone)
- `PLAYBOOK.md` — long DIY walkthrough for non-technical user
- `PROMPTS.md` — Claude Code prompts (copy-paste ready)
- `INTEGRATIONS.md` — extra prompts for Google Sheets, Whop, OAuth
- `QUICKSTART.md` — minimal setup for developers
- `src/App.jsx` — the entire dashboard
