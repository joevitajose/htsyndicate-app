# Quick Start (5 minutes)

If you just want to see the dashboard running locally first:

```bash
# 1. Install Node.js if you don't have it
# Download from https://nodejs.org

# 2. In this folder, install dependencies
npm install

# 3. Start the dev server
npm run dev

# 4. Open the URL it gives you (usually http://localhost:5173)
```

That's it. The dashboard will load with all demo data.

For deployment to a real working app, see README.md.
